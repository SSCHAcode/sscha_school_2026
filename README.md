# Tutorials and materials for the SSCHA online school 2026

This repository cointains two directories:
1. `Materials`
2. `Tutorials`

Inside `Materials` you will find the required files to run each tutorial, plus a machine learning force-field `GAP_1.xml` with related files of CsPbI3 perovskite.
This force field is trained for this school and should not be used for research purpouses.
If you need to use it for research purpouses, remember to contact the authors and the organizers of the school for instructions on how to properly cite it.

The scripts provided and the tutorials themselves are released under the GPLv3 license, and can be employed within the rights of the license (see the LICENSE file). 
Remember that the SSCHA code itself runs on the GPLv3, and if used for research or commercial product, it should be disclosed and appropriately cited (see www.sscha.eu, on References and how to cite).
